# 🎬 Sistema de Gestión de Personajes - Teatro de Estrellas

Este proyecto es una aplicación web interactiva diseñada con una estética inspirada en el cine y los premios Óscar. Permite gestionar y organizar una cartelera de personajes dinámicos de forma ágil y atractiva.

## 🚀 Características Principales

- **Gestión Completa de Personajes (CRUD):**
  - **Crear:** Añade nuevos personajes con nombre, edad, altura, imagen personalizada, descripción, habilidades y clasificación (Héroe o Villano).
  - **Visualizar:** Tarjetas interactivas con detalles completos y etiquetas diferenciadas para Héroes y Villanos.
  - **Editar:** Modifica los atributos de cualquier personaje de forma dinámica mediante un formulario integrado.
  - **Eliminar:** Quita personajes de la lista con confirmación de seguridad.
- **Filtros y Ordenamiento:**
  - Ordena la cartelera alfabéticamente por **Nombre**.
  - Ordena la cartelera de menor a mayor por **Edad**.
- **Estética Cinemática:**
  - Fondo animado con temática de premios Óscar.
  - Interfaz responsiva con un diseño moderno, tarjetas estilizadas y efectos de transición suaves.
